# Jenny

A Python based Mirai honeypot built for easy setup & use.

## Requirements

* Python 2.7

## Usage

```
python jenny.py
```

## Future features

* Web interface for live output
* Mitigate false-positives
* XML Logs

## Known issues

* socket.error: [Errno 98] Address already in use
```
Solution: sudo fuser -k 23/tcp
```
